# Demo Video Assets

Place these 4 video files in this directory:

- `campus_chase.mp4` — Chasing scenario (CCTV-style footage)
- `hostel_fall.mp4` — Fall detection scenario
- `parking_struggle.mp4` — Struggle/fight scenario
- `metro_following.mp4` — Following/stalking scenario

## Guidelines

- **Duration**: 15-30 seconds each (videos will loop)
- **Style**: CCTV / surveillance camera footage
- **Format**: MP4 (H.264 recommended for browser compatibility)
- **Resolution**: 720p or higher recommended

If videos are not present, the camera grid will show a dark placeholder with the camera name.
